# 📘 Google Workspace MCP Servers in Antigravity 2.0

1. Introduction
This codelab teaches you how to use Model Context Protocol (MCP) to connect your Google Workspace data to Antigravity 2.0, using either the IDE or CLI.

<img src=".././Images/mcp_workspace.png" width="600" height="300">

## Goal: 
**You will enable secure, code‑free access to:**
- Gmail
- Google Drive
- Google Calendar
- Google Chat
- People API
MCP allows Antigravity to use these Workspace tools through remote servers without writing any integration code. 

2. What You’ll Do
According to the codelab, you will complete three tasks: 

Enable Google Workspace MCP services in your Google Cloud project.

Configure Antigravity (IDE or CLI) to access these MCP services.

Test the integration using prompts that interact with your Workspace data.

## 3. What we Need
The codelab lists the following prerequisites: 
- A web browser (e.g., Chrome)

A Google Cloud project enrolled in the Google Workspace Developer Preview Program

A Google Workspace account

Antigravity 2.0 IDE and/or CLI installed on your machine
